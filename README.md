# ✅ 아두이노 + R9D를 이용한 LED 제어 시스템

본 프로젝트는 **R9D 수신기와 아두이노 보드**를 이용하여 총 3가지 기능을 구현한 시스템입니다:

1. **LED On/Off**
2. **LED 밝기 제어**
3. **3색(RGB) LED 색상 변환**
